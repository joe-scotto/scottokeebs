# Scotto48 (PCB Edition) (SK44)

The Scotto48 (PCB Edition) is an ultra low-profile 48-key Choc keyboard with an exposed controller and diodes.

You can read more about the components and how to build it [here](https://scottokeebs.com/blogs/keyboards/scotto48-pcb-keyboard).
