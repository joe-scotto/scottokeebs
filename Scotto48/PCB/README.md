# Scotto48 (PCB Edition) (SK44)

The Scotto48 (PCB Edition) is a super low-profile 34-key split monoblock ortholinear keyboard that uses 18x17mm spaced Choc switches and an exposed controller in the middle.

You can read more about the components and how to build it [here](https://scottokeebs.com/blogs/keyboards/scotto48-pcb-keyboard).
