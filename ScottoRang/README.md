# ScottoRang (SK40)

The ScottoRang is a 34-key split monoblock ergonomic 16x16mm Choc spaced keyboard with a central 128x64 OLED display.

You can read more about the components and how to build it [here](https://scottokeebs.com/blogs/keyboards/scottorang-handwired-keyboard).
