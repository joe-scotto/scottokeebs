# ScottoNum (SK45)

The ScottoBigum is a 17-key wireless macropad scaled up 4 times the normal size and arguably a world record.

You can read more about the components and how to build it [here](https://scottokeebs.com/blogs/macropads/scottobigum-handwired-macropad).
